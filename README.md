## Class Person
Данный класс имеет 2 свойства имя и дата рождения 
Дата рождения имеет тип данных DataTime 
Также внутри есть метод IsOnlyLetters который проверяет
Есть ли в строке что то кроме пробелов и английских букв

Буквами и ничего более ни цифр никаких других символов кроме
